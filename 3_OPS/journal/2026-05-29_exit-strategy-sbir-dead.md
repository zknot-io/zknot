# Exit Strategy Analysis + SBIR Pivot

## TL;DR

Tested the question "can ZKNOT be sold for $5M in 6 months?" against the full
patent portfolio (16+ provisionals, all orbiting one idea: cryptography gated by
verified physical reality). Verdict: a stretch but not fantasy — IF the goal is
reframed from "sell the company" to "engineer a strategic acquisition of ONE
capability + IP priority dates + the SDVOSB vehicle, by a buyer with money AND
urgency." That buyer is in defense, not journalism. The wedge is the
authenticated-heartbeat-loss -> autonomous key-erasure patent (captured-drone
anti-tamper), optionally bundled with silicon-to-sample CBRNE sensor binding.

Separately and importantly: the SBIR is DEAD for now. The solicitation that fit
no longer applies unless ZKNOT completely changes course. It is removed as an
on-ramp and as a milestone. Its loss matters because it was carrying the
"third-party validation signal" role in the exit plan — that role now shifts
onto a prime LOI/eval and the working demo.

## Context

Single working session. Uploaded DOC-003 (all-patents summary) and asked for a
6-month / $5M exit thesis. Mid-session, killed SBIR from the plan and from
memory. This entry exists to survive deletion of the source thread.

## What buyers actually pay $5M for

Three things, and only three:
1. Revenue (can't manufacture in 6 months — out).
2. Team + a must-have capability (acqui-hire flavor).
3. An urgent strategic gap the IP plugs (this is the play).

Provisional patents alone almost never clear $5M — a buyer is purchasing a
priority date plus a conversion bill, not examined claims. So the value has to
come from urgency + a demonstrable capability + a transferable federal vehicle.

## Why the drone/anti-tamper wedge over the evidence-integrity story

The evidence-integrity vision (journalists, election observers, legal
chain-of-custody) is the emotionally compelling narrative but it's SLOW money:
poor buyers, long cycles, small budgets. Wrong fuel for a 6-month exit.

The heartbeat-loss -> key-erasure patent (filed March 2026) targets a live,
funded DoD problem (captured-asset anti-tamper). Buyer pool — defense primes,
counter-UAS shops — moves fast and pays real money. Scope is narrow enough to
actually DEMO. Everything else in the portfolio becomes "and the platform
extends to..." data-room upside, not the headline.

## Decisions

- **DECISION: Reframe the goal.** Not "sell ZKNOT." Engineer a strategic
  acquisition of one capability + IP priority dates + SDVOSB vehicle by a
  defense buyer with urgency.
- **DECISION: The wedge is heartbeat-loss -> autonomous key erasure**, optionally
  bundled with silicon-to-sample CBRNE sensor binding. All other patents are
  data-room upside, not the pitch.
- **DECISION: Target buyer pool is defense** (primes + counter-UAS), NOT
  journalism / civil evidence markets.
- **DECISION: SBIR is dead for now.** Removed as on-ramp and milestone. Do not
  resurrect unless course changes completely. (Recorded to memory this session.)
- **DECISION: Demo + one prime relationship are now load-bearing**, not
  "nice to have," because the SBIR validation signal is gone.
- **OPEN / NOT YET DECIDED: Whether to file non-provisional vs PCT, and exactly
  which 2-3 patents to convert.** Needs a patent attorney. Do not DIY at this
  stake.

## The 6-month sequence (as planned this session)

1. Weeks 0-4: Hire patent attorney; convert 2-3 crown jewels provisional ->
   non-provisional/PCT. The March-2026 heartbeat-erasure app has a hard ~12-mo
   clock — for SALE value, file well before the deadline so buyer counsel can
   diligence real claims.
2. Weeks 4-12: Build a HARD demo — working bench rig, real silicon: signed
   authenticated heartbeat -> consecutive-miss counter -> secure-element key
   erasure -> signed terminal erasure record. Film it. (Bench EE/firmware work,
   not an AI task. This single artifact moves valuation most.)
3. Weeks 8-16: Manufacture a demand signal — one LOI / NDA'd eval / pilot
   interest from a prime or program office. Changes the number 5-10x. (Founder
   relationship work.)
4. Weeks 12-20: Run the process — boutique IP/M&A advisor or targeted corp-dev
   outreach. Clean data room: filed claims, demo video, SDVOSB cert, SAM reg.
5. Weeks 20-26: Close. Expect structure, not a clean wire — $5M is plausibly
   $2-3M cash + earnout + retention. Cash-only at $5M in 6 mo would be
   exceptional.

## Validation signal — the gap the dead SBIR leaves

SBIR was the cheap "someone official bet on this" signal. Substitutes, strongest
first:
- A prime's NDA'd technical eval or LOI (now the single most valuable artifact;
  carries more weight than an SBIR anyway, just harder to get).
- A non-SBIR funding/validation vehicle — OTA, prime IRAD interest, AFWERX/DIU
  non-SBIR pathways. Some move faster than SBIR did. [NEEDS RESEARCH — which are
  open right now.]
- The working demo itself — now carries more of the burden of proof on its own.

## Where Claude is useful vs not (honest split)

- Claude CAN: draft non-provisional claim language (for attorney to refine),
  build teaser/CIM and data-room docs, size the market, write prime outreach,
  code the demo's host software.
- Claude CANNOT: build the hardware demo, do the legal conversion, carry prime
  relationships, run the negotiation. Those four are the actual binding

## Personal monopoly note

The real monopoly is the INTERSECTION: hardware-rooted physical-presence
cryptography x veteran-owned federal access x a coherent patent thicket no one
else is assembling. The 6-month exit is just monetizing that intersection before
it's competed away.

# ZKKey Connect

Signing device. ATECC608B + RP2350 (Pico Plus 2 module).

## Revisions
- **rev0-breadboard** — pre-PCB validation, breakouts on breadboard
- **rev1-pico-carrier** — first custom PCB, Pico Plus 2 socketed

## Future (not yet started)

- rev3: production polish, panelization, test-point access
